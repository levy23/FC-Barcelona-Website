piłkarskiemu FC Barcelona. Zbiór najświeższych danych,
statystyk i informacji klubowych. Strona edytowalna przez
zarejestrowanych użytkowników.

html/CSS/LESS/Gulp/PHP/JavaScript/bootstrap/RWD
